# Pâtisserie à Domicile

Site web de pâtisserie avec système de réservation en ligne, gestion d'ateliers et administration.

## Technologies

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4** avec couleurs personnalisées
- **Zod** pour la validation
- **Prisma** pour la base de données  
- **Nodemailer** pour les emails

## Couleurs du site

- **Primaire**: Vert pomme - #8DB600
- **Secondaire**: Sucre roux - #B8860B

## Démarrage

```bash
npm install
cp .env.example .env
npm run dev
```

Ouvrez http://localhost:3000
